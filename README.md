SPÉCIFICATION FONCTIONNELLE – SITE WEB KPOP IN LYON (V1)

1. OBJECTIF DU SITE

Le site a pour objectif de promouvoir et organiser des événements K-pop, en mettant en avant une expérience visuelle immersive, moderne et centrée sur l’utilisateur.

Le site doit être :

- Mobile first
- Très visuel (images dominantes, overlays légers)
- Design moderne inspiré de l’univers K-pop (couleurs pop, typographie impactante)
- Navigation simple et rapide (4 menus principaux)

Menus principaux :

- Événement
- Avis
- À propos
- Contact

---

2. PAGE D’ACCUEIL (HOME)

2.1 Structure globale

La page d’accueil contient un système de slides (carousel) en plein écran ou quasi plein écran.

Chaque slide est centré, avec un overlay léger sur les images pour assurer la lisibilité du texte.

---

2.2 Slide 1 : Événement à venir

Objectif : Mettre en avant l’événement principal.

Contenu :

- Titre : "Événement à venir"
- Visuel principal : grande affiche (image immersive)
- Texte overlay :
  - Titre principal : "SEVENTEEN EVENT"
  - Date (ex : May 19, 2026)
  - Lieu (ex : Lyon)
  - Description courte : "Fan event dedicated to SEVENTEEN"
- Call To Action (CTA) :
  - Bouton : "Join Event"

Comportement :

- Le bouton redirige vers la page Événement
- Animation légère (hover / transition)

---

2.3 Slide 2 : Événements passés

Objectif : Valoriser les anciens événements et créer de la crédibilité.

Contenu :

- Titre : "Past Events"
- Texte :
  "Thank you for all the amazing moments shared together."
- Visuels :
  - 1 à 3 images maximum d’événements passés
- Informations associées :
  - Note moyenne (calculée à partir des avis utilisateurs)
  - Nombre de participants

Comportement :

- Les visuels peuvent être affichés en grille ou carousel
- Les notes sont affichées sous forme d’étoiles ou score numérique

---

3. PAGE ÉVÉNEMENT

Objectif : Présenter l’événement actuel en détail.

Contenu :

- Titre principal : "SEVENTEEN EVENT"
- Date : "May 19, 2026"
- Lieu : "Lyon"
- Description :
  "A fan event dedicated to SEVENTEEN, bringing fans together to celebrate their universe."

Design :

- Image de fond immersive
- Texte centré avec overlay
- Style impactant et minimaliste

---

4. PAGE AVIS

Objectif : Afficher les retours des participants.

Contenu :

- Liste d’avis utilisateurs :
  - Nom ou pseudo
  - Note (étoiles ou score)
  - Commentaire court

Fonctionnalités :

- Bouton : "Laisser un avis"
- Possibilité d’ajouter un avis (formulaire à prévoir ultérieurement)

Affichage :

- Cartes d’avis (cards)
- Design épuré et lisible

---

5. PAGE À PROPOS

Objectif : Présenter le projet.

Contenu :

"Kpop in Lyon est un projet créé par des passionnés de K-pop.
Nous organisons des événements dédiés à des artistes et groupes afin de rassembler les fans autour de moments uniques.
Notre objectif est de proposer des expériences immersives et conviviales."

Design :

- Texte centré
- Visuel d’ambiance en fond possible
- Style simple et authentique

---

6. PAGE CONTACT – COLLABORATION FAN BASE

Objectif : Permettre aux fanbases de proposer une collaboration.

Titre :
"Collaboration Fan Base"

Introduction :
"Organiser un événement avec Kpop in Lyon.
Vous êtes une fan base et vous souhaitez collaborer avec nous autour d’un événement K-pop ?
Remplissez ce formulaire afin que nous puissions étudier votre projet."

---

6.1 Formulaire

6.1.1 Section : Infos de base

- Nom prénom (champ texte)
- Nom de la fanbase (champ texte)
- Email (champ email)
- Réseaux sociaux (champ texte)

---

6.1.2 Section : Projet

- Groupe / artiste (champ texte)
- Ville / lieu (champ texte)
- Date ou période (champ texte ou date)

---

6.1.3 Section : Type de collaboration (choix multiples)

- Organisation complète par Kpop in Lyon
- Co-organisation avec la fan base
- Collaboration avec fourniture d’éléments (visuels, freebies, lots)
- Collaboration visuelle (repartage uniquement)
- Aide à la promotion

---

6.1.4 Section : Décorations (choix multiples)

- Décoration fournie par Kpop in Lyon
- Éléments fournis par la fan base
- Décoration collaborative
- Décoration immersive adaptée à l’univers de l’artiste

---

6.1.5 Section : Détails

- Champ texte libre :
  "Décrivez votre projet, vos idées ou attentes."

---

6.1.6 Soumission

- Bouton : "Envoyer la demande"

Comportement :

- Validation des champs obligatoires
- Envoi des données (API ou email à définir)

---

7. EXIGENCES UI/UX

- Mobile first obligatoire
- Contenu centré
- Images dominantes
- Overlay léger pour lisibilité
- Texte court et impactant
- Animations fluides et modernes
- Univers visuel K-pop (couleurs pop, dynamique, immersif)
- Thème / couleur de style kpop boys band, classe
